# Sales Performance Dashboard

## Project Overview

Developed an interactive Power BI dashboard using the Superstore Sales Dataset to analyze sales performance across regions and product categories.

## Dataset Information

- 9,800+ Sales Records
- 4 Regions
- 3 Product Categories

## KPIs

- Total Sales
- Total Orders
- Total Customers
- Average Sales

## Dashboard Features

- Sales by Region Analysis
- Sales by Category Analysis
- Monthly Sales Trend
- Top 10 Products by Sales
- Interactive Region and Category Filters

## Tools Used

- Power BI
- Power Query
- Microsoft Excel

## Key Insights

- Technology category generated the highest sales.
- East and West regions contributed significantly to overall revenue.
- Top-performing products were identified using sales analysis.
